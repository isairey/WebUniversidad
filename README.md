# 🎓 Universidad Web

Sitio web institucional diseñado para representar una universidad, proporcionando información académica, administrativa y de servicios para estudiantes, docentes y visitantes.

---

## 🚀 Descripción

**Universidad Web** es una página desarrollada con tecnologías web básicas que permite mostrar información relevante como carreras, servicios, noticias y contacto.

El sistema está enfocado en ofrecer una **experiencia clara, organizada y accesible** para todos los usuarios.

---

## 🎯 Objetivos

- Presentar información institucional de forma clara  
- Facilitar el acceso a servicios académicos  
- Mejorar la comunicación con estudiantes  
- Ofrecer una interfaz moderna y responsive  

---

## ✨ Características

### 🏫 Información Institucional
- Historia de la universidad  
- Misión, visión y valores  
- Información general  

### 📚 Oferta Académica
- Carreras disponibles  
- Descripción de programas  
- Planes de estudio  

### 📢 Noticias y Eventos
- Publicaciones recientes  
- Eventos académicos  

### 📞 Contacto
- Formulario de contacto  
- Información de ubicación  

### 📱 Diseño Responsive
- Compatible con móvil, tablet y escritorio
